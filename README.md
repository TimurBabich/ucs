# ucs
Приложение планируется использовать для валидации пользователей.
На текущий момент выводит информацию о пользователе в формате JSON из базы данных mysql по логину и паролю. 
Работа с mysql производится через Hibernate.
Для построения проекта используется Maven.
Задействован Spring MVC.
В тестовых целях используется Jetty.
